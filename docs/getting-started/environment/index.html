Environment

The SDK will target an environment based on the useSandbox value you provide when you initialize it:

Sandbox (true): https://api.sandbox.checkout.com

Live (false): https://api.checkout.com

caution

Public and secret key pairs will only work for their relevant environment. Sandbox keys begin with "pk_test_" or "sk_test_". You will receive separate Live keys for use in your production environment when going live with Checkout.com.

Testing#

If you are testing in the Sandbox environment, Checkout.com provides a list of test card details. You can also simulate a lot of edge cases like Declines, by using special transaction values.

note

You can also test 3D Secure with the test cards provided. If you want to test 3DS 2 flows, use the cards mentioned here.
